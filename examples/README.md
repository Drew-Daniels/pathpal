# PathPal Examples

These examples demonstrate PathPal's features. Run them from the project root.

## Prerequisites

Build the project first:

```bash
npm run build
```

## Examples

### Basic Usage

Demonstrates path generation with nested directories:

```bash
node examples/basic.js
```

### File Operations

Creates a temporary directory structure and demonstrates read/write/list operations:

```bash
node examples/file-operations.js
```

### Project Structure

Shows a realistic web application directory structure with PathPal:

```bash
node examples/project-structure.js
```

### Security

Demonstrates strict mode and filename sanitization:

```bash
node examples/security.js
```
